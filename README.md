# Libraverse - Public API CI4

Aplikasi pencarian dan pembacaan buku menggunakan framework CodeIgniter 4 dan Gutendex Public API.

## Fitur
- Search buku
- Detail buku
- Baca buku langsung
- Responsive UI
- Animasi scroll
- Reader buku di dalam website

## Teknologi
- CodeIgniter 4
- Bootstrap 5
- Gutendex API
- HTML
- CSS
- JavaScript

## Public API
https://gutendex.com/
